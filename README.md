# Carrito de Compras - Pan Viajero

Proyecto personal para una amiga. Un formulario de venta basado en un carrito de compras.

## Tecnologías

- Vue 3
- Vuex
- Vite
- TailwindCSS

## Configuración

Para ejecutar el código y llevarlo a producción:

### Primero instalar dependencias

```
npm install
```

### Para desarrollo

```
npm run dev
```

### Para producción

```
npm run build
```
